# EduGuard-AI

AI-powered educational analytics and early-warning system using Explainable AI, behavioural personas, intervention intelligence, and GenAI support for student risk prediction.

## Features

- Student risk prediction using XGBoost
- Early-warning system
- Behavioural persona analytics
- Explainable AI using SHAP
- AI intervention recommendation engine
- Psychological engagement analysis
- FastAPI backend
- Interactive dashboard frontend

## Tech Stack

- Python
- XGBoost
- Scikit-learn
- SHAP
- FastAPI
- React
- Tailwind CSS
- Bolt.new
